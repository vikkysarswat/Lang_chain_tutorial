## ğŸ› Troubleshooting

### Common Issues

**"Module not found" errors**
```bash
pip install -r requirements.txt
```

**"API key not found"**
```bash
export OPENAI_API_KEY='your-key-here'
```

**"Rate limit exceeded"**
- Add delays between API calls
- Implement caching
- Use smaller models

**"Out of memory"**
- Reduce chunk sizes
- Process documents in batches
- Use streaming for large outputs
